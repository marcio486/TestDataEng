Todos os painéis obedecem os seguintes filtros -> DATA, SEGMENTO, ESTADO, CIDADE E NOME
Em todos os painéis é possível filtrar qualquer valor que seja desejado para análise.

*Os dados a partir do mês 08/2019 são valores ainda a serem recebidos (pois não existem novas contratações após 07/209)

* Gráficos das páginas 4 ~ 6 possuem a mais de uma métrica por visualização, para acessá-las basta selecionar a métrica desejada no botão superior direito "Metrícas opcionais"

Painel 1/6  -> Resultados Gerais
1.Tabela 1  -> Resultado gerais possuindo todos os meses com os dados MRR,Contraction,Canceled,Expansion,Ressurection,Ativos,Bronze,Prata,Ouro e platina
2.Gráfico MRR -> Evolução de MRR através dos meses
3.Gráfico ATVIOS -> Evolução de usuários ativos e com plano bronze/prata/ouro/platina através dos meses

Painel 2/6  -> Resultados Expansion/Ressurection/Cancelled/Contraction
1.-> Evolução das métricas Expansion/Ressurection/Cancelled/Contraction atráves dos mêses para facilitar a visualização
2.-> Evolução das métricas Expansion/Ressurection/Cancelled/Contraction atráves dos mêses em um único gráfico para facilitar comparações

Painel 3/6  -> Resultados Gerais Estado/Cidade
1.-> MRR por estado
2.-> Ativos por estado
3.-> Expansion/Ressurection/Cancelled/Contraction por estado
4.-> bronze/prata/ouro/platina por estado

Painel 4/6  -> Resultados Gerais Estado/Cidade 2
1.-> Evolução da métrica MRR/Expansion/Ressurection/Cancelled/Contraction (inicialmente é apresentado somente a métrica MMR, sendo possível selecioar as outras através das "Metrícas opcionais") por mês comparando múltiplos estados. Podem ser comparados no máximo 5 estados ao mesmo tempo, mais linhas dificultam a visualização dos dados (inicialmente os 5 estados com maior MMR serão exibidos)
2.-> Evolução de usuários Ativo/Bronze/Prata/Ouro/Platina (inicialmente é apresentado somente os usuários ativos, sendo possível selecionar os outros planos através das "Metrícas opcionais") por mês comparando múltiplos estados. Podem ser comparados no máximo 5 estados ao mesmo tempo, mais linhas dificultam a visualização dos dados (inicialmente os 5 estados com maior numero de usuários ativos serão exibidos)


Painel 5/6  -> Resultados Gerais por segmento
1.-> Distribuição das métricas MRR/Canceled/Ressurection/Contraction ("Métricas opcionais") por segmento
2.-> Comparação da distribuição das métricas Canceled/Ressurection/Contraction por segmento
3.-> Evolução de clientes Ativos/Bronze/Prata/Ouro/Platina ("Métricas opcionais") por segmento

Painel 6/6  -> RResultados Gerais por segmento 2
1.-> Evolução da métrica MRR/Expansion/Ressurection/Cancelled/Contraction (inicialmente é apresentado somente a métrica MMR, sendo possível selecioar as outras através das "Metrícas opcionais") por mês comparando múltiplos segmentos. Podem ser comparados no máximo 5 segmentos ao mesmo tempo, mais linhas dificultam a visualização dos dados (inicialmente os 5 segmentos com maior MMR serão exibidos)
2.-> Evolução de usuários Ativo/Bronze/Prata/Ouro/Platina (inicialmente é apresentado somente os usuários ativos, sendo possível selecionar os outros planos através das "Metrícas opcionais") por mês comparando múltiplos segmentos. Podem ser comparados no máximo 5 segmentos ao mesmo tempo, mais linhas dificultam a visualização dos dados (inicialmente os 5 segmentos com maior numero de usuários ativos serão exibidos)
